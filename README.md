# **✨Dashboard Analisis E-Commerce**

## **Deskripsi**
Aplikasi dashboard interaktif menggunakan Streamlit yang menyajikan analisis data e-commerce yang mencakup total transaksi, revenue, dan hubungan antara review score dengan total revenue produk selama periode Januari - Desember 2018.

## **Setup Environment - Anaconda**
```bash
conda create --name main-ds python=3.13.9
conda activate main-ds
pip install -r requirements.txt
```

## **Setup Environment - Shell/Terminal**
```bash
mkdir Submission_Hana Talitha Syahda
cd Submission_Hana Talitha Syahda
pipenv install
pipenv shell
pip install -r requirements.txt
```

## **Run streamlit app**
```bash
streamlit run dashboard.py
Local URL: http://localhost:8503
```
